🏖️ Beautiful Beaches

Hệ thống quản lý và đặt tour du lịch bãi biển hiện đại, thân thiện người dùng.

Nhóm: BeachTeam-2025

Ngày cập nhật: 01/07/2025

Ngôn ngữ: Laravel + JS + Bootstrap

Thành viên: Nguyễn Duy Tân, Phan Văn Hiếu, Phan Thị Phương Trúc, Phạm Thanh Nhân

Giới thiệu dự án

Beautiful Beaches là một hệ thống web quản lý và đặt tour du lịch bãi biển, cho phép người dùng khám phá, đánh giá, đặt tour, quản lý thông tin cá nhân, và cung cấp các chức năng quản trị cho admin, CEO. Dự án sử dụng framework Laravel (PHP) kết hợp với giao diện hiện đại, thân thiện, hỗ trợ nhiều vai trò người dùng.

Mục tiêu
  • Xây dựng nền tảng đặt tour bãi biển chuyên nghiệp, dễ sử dụng.
  • Hỗ trợ quản lý đa vai trò: Admin, CEO, User.
  • Tích hợp hệ thống đánh giá, bình luận, báo cáo doanh thu, quản lý tour, bãi biển, người dùng.

Kiến trúc hệ thống

  • Backend: Laravel 12.x, PHP 8.2+
  • Frontend: Blade Template, Bootstrap, CSS/JS tùy chỉnh
  • Database: SQLite (có thể chuyển sang MySQL)
  • Authentication: Laravel Auth, phân quyền theo middleware
  • Cấu trúc thư mục: Theo chuẩn Laravel MVC
  • Quản lý source: Git, chia branch theo chức năng

Sơ đồ tổng quan:

3.1 Quản lý người dùng

  • Đăng ký, đăng nhập, đăng xuất
  • Phân quyền: admin, ceo, user
  • Quản lý thông tin cá nhân, đổi mật khẩu
  • Admin có thể thêm, sửa, xóa, ban user

3.2 Quản lý bãi biển

  • Thêm, sửa, xóa bãi biển (admin)
  • Quản lý chi tiết bãi biển: tên, khu vực, hình ảnh, mô tả, tags
  • Hiển thị danh sách bãi biển cho người dùng

3.3 Quản lý tour du lịch

  • Thêm, sửa, xóa tour (CEO)
  • Quản lý chi tiết tour: tên, giá, thời gian, bãi biển liên quan, dịch vụ kèm theo
  • Hiển thị danh sách tour, lọc theo khu vực, tìm kiếm

3.4 Đặt tour

  • Người dùng đặt tour, nhập thông tin liên hệ
  • Quản lý lịch sử đặt tour (user)
  • CEO xác nhận, cập nhật trạng thái booking

3.5 Đánh giá & Bình luận

  • Người dùng đánh giá, bình luận bãi biển/tour
  • Hiển thị danh sách đánh giá, xếp hạng sao
  • Quản lý, kiểm duyệt bình luận (admin/ceo)

3.6 Thống kê & Báo cáo

  • Thống kê doanh thu theo tháng (CEO)
  • Báo cáo số lượng booking, doanh thu, tour nổi bật
  • Giao diện báo cáo trực quan, xuất file

3.7 Trang thông tin & liên hệ

  • Trang giới thiệu, gallery, liên hệ, gửi thắc mắc
  • Footer, header, sidebar tùy biến theo vai trò

📁 Mô tả các thư mục

  • Controllers: Xử lý logic của từng module: User, Tour, Booking, Report...
  • Middleware: Kiểm soát quyền truy cập, kiểm tra đăng nhập.
  • Models: Đại diện bảng dữ liệu như User, Tour, Beach...
  • views/: Blade template chia theo vai trò: admin, ceo, user.
  • migrations: Tạo bảng CSDL.
  • seeders: Thêm dữ liệu mẫu ban đầu.
  • web.php: Định nghĩa route toàn bộ hệ thống.
  • public/: Chứa ảnh, JS, CSS truy cập công khai.
  • config/: Cấu hình app, DB, mail,...
  • artisan: CLI của Laravel hỗ trợ tạo file nhanh.
  • .env: Biến môi trường riêng cho dự án.
  • README.md: Hướng dẫn sử dụng, giới thiệu dự án.

5. Hướng dẫn cài đặt & chạy dự án

📦 Các bước cài đặt:

1. Clone source về máy:

<?php
public function index()
{
  $reviews = Review::all();
  return response()->json($reviews);
}

2. Cài đặt thư viện PHP:

composer install

3. Cài đặt thư viện JS (nếu dùng):

npm install npm run dev

4. Tạo file cấu hình môi trường:

cp .env.example .env php artisan key:generate

5. Cấu hình database trong .env:

  • Mặc định dùng SQLite: DB_CONNECTION=sqlite
  • Nếu dùng MySQL, sửa lại các thông tin kết nối

6. Tạo database và migrate:

php artisan migrate php artisan db:seed

7. Chạy server:

php artisan serve

🔗 Truy cập http://127.0.0.1:8000

6. Hướng dẫn sử dụng các chức năng

6.1 Đăng ký, đăng nhập, phân quyền

  • Người dùng đăng ký tài khoản và đăng nhập để sử dụng các chức năng như đặt tour, đánh giá, quản lý cá nhân.
  • Admin và CEO đăng nhập để truy cập các chức năng quản trị hệ thống.

6.2 Quản lý bãi biển

  • Admin truy cập dashboard, chọn mục “Quản lý bãi biển”.
  • Thực hiện các thao tác: thêm mới, chỉnh sửa, xóa bãi biển.
  • Thông tin mỗi bãi biển gồm: tên, khu vực, hình ảnh, mô tả, tags liên quan.

6.3 Quản lý tour

  • CEO vào dashboard, chọn mục “Quản lý tour”.
  • Thêm/sửa/xóa tour du lịch: nhập tên tour, giá, thời gian, bãi biển liên quan và dịch vụ kèm theo.

6.4 Đặt tour

  • Người dùng chọn tour mong muốn từ danh sách.
  • Điền thông tin liên hệ và xác nhận đặt tour.
  • Xem lại lịch sử đặt tour tại trang dashboard cá nhân.

6.5 Đánh giá & bình luận

  • Người dùng vào chi tiết bãi biển hoặc tour để đánh giá.
  • Gửi số sao (rating) và bình luận nội dung.
  • Đánh giá hiển thị công khai và có thể được kiểm duyệt bởi admin/CEO.

6.6 Thống kê & báo cáo

  • CEO vào mục “Thống kê doanh thu” để theo dõi số liệu hệ thống.
  • Xem báo cáo doanh thu, số lượng booking, tour nổi bật.
  • Có thể lọc theo tháng, năm và xuất ra file báo cáo.

6.7 Trang thông tin & liên hệ

  • Người dùng có thể truy cập các trang: Giới thiệu, Gallery, Liên hệ, Gửi thắc mắc.
  • Form liên hệ sẽ gửi thông tin đến quản trị viên để xử lý yêu cầu của người dùng.

8. Phụ lục: Một số đoạn code mẫu

8.1 Migration tạo bảng reviews

<?php
Schema::create('reviews', function (Blueprint $table) {
  $table->id();
  $table->foreignId('user_id')->constrained()->onDelete('cascade');
  $table->unsignedBigInteger('beach_id');
  $table->tinyInteger('rating')->unsigned();
  $table->text('comment')->nullable();
  $table->timestamps();
  $table->foreign('beach_id')->references('id')->on('beaches')->onDelete('cascade');
});

8.2 Model Review

<?php
class Review extends Model
{
  use HasFactory;
  protected $fillable = ['user_id', 'beach_id', 'rating', 'comment'];
}

8.3 Controller trả về danh sách review

<?php
public function index()
{
  $reviews = Review::all();
  return response()->json($reviews);
}
